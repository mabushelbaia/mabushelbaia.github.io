# Personal Website

Source for [mabushelbaia.com](https://mabushelbaia.com) — a single, self-contained
static page (`index.html`) with inline CSS and JS. No build step or dependencies.

## Structure

- `index.html` — the site (all markup, styles, and scripts inline)
- `mabushelbaia_cv.pdf` — published CV, linked from the site
- `cv/` — LaTeX source for the CV (see `cv/README.md`)
- `CNAME` — custom domain for GitHub Pages

Just open `index.html` in a browser, or serve the folder with any static server.
